# 快捷键

## iterm2快捷键

```
新建标签：Cmd + t

关闭标签：Cmd + w

切换标签：Cmd + 数字 Cmd + 左右方向键

切换全屏：Cmd + enter

垂直分屏：Cmd + d

水平分屏：Cmd + shift + d

搜索文本，复制(Cmd + F, Tab)

剪切板历史记录 (Cmd + Shift + H)

ctrl + u 清除当前行
ctrl + a 到行首
ctrl + e 到行尾
ctrl + w 删除光标之前的单词
ctrl + k 删除到文本末尾
```
